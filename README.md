## About
Querying data from a database is a very common skill in data science. Structured Query Language, abbreviated SQL, is one of the most common methods of pulling information from databases for further analysis.

In the context of the data science workflow, SQL is most commonly used when obtaining data, but can also be used for exploratory data analysis.

This lab involves analyzing the Basketball (`ncaa_basketball`) data set in Google BigQuery to answer specific questions.

**Data set:** `bigquery-public-data.ncaa_basketball`

## Submission
Lab submissions should be made via Canvas. Your submission will be in the form of a SQL script (`.sql`) that contains the questions and your queries to answer them.  You should include a link to your Github Enterprise fork of this repository, and put your script(s) in there.
